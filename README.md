# To-Do App

A simple and responsive To-Do List application built using HTML, CSS, and JavaScript.

## Description

This project helps users organize daily tasks by allowing them to add, complete, and delete tasks through a clean and user-friendly interface.

## Features

- Modern Glassmorphism UI
- Light / Dark Mode
- Dynamic Progress Bar
- Real-time Search
- Task Filtering
- Smart Sorting
- Clear Completed Tasks
- LocalStorage Integration

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
